pi coding agent configuration

This repository contains configuration, extensions, themes, and settings for the `pi` coding agent used by the developer's environment. It is not an application; it provides the agent with:

- custom extensions and shortcuts
- keybindings and themes
- project-global agent instructions (AGENTS.md)

Quick notes
- Keep secrets out of the repo (e.g. auth.json).
- Edit AGENTS.md to tweak assistant style and rules.

How to use
1) Open this folder with the pi coding agent.
2) The agent reads AGENTS.md and the extensions/ and themes/ folders on startup.
